Beowolf Cluster at AU

After doing some research, I was not able to find updated information in order to build this project. In doing projects like these documention is helpful. So I have created thiss repository.

The Steps I took

  1. Install Debian off campus cause eagle secure sucks
    • This makes you create a separate non-root user
    • Automatically logs into new user
  2. Make sure your master node has an IP address
  3. Login as root:
    • $ su root
    • Type password
  4. Install sudo:
    • Apt-get install sudo
  5. Give sudo access to admin user in order to disable root log-in
    • Run $ visudo command
    • Look for
      # User privilege specification
      root ALL=(ALL:ALL) ALL
      newuser ALL=(ALL:ALL) ALL #ADD THIS LINE
  6. New user should have admin privileges now, no need to use root now
  7. Su username //switches to new user, will ask for password
  8. Create SSH login
    • $ apt-get update //not necessary but just in case
    • $ sudo apt-get install openssh-server
    • $ ssh-keygen -t dsa //generates a new ssh key
      • Press enter to keep default folder location
      • Leave passphrase empty, just press enter to skip
  9. Now you will have the .ssh folder in your home directory
    • Copy key to a new folder authorized_keys

      $ cd /home/user/.ssh; cat id_dsa.pub >> authorized_keys;
  10. Download MPICH
    • I used the Synaptic Package Manager to install this, choose the latest MPICH version
  11. Check for sshd
